# Machine Learning Guidelines for Agents

## Dev environment tips

- Use `uv` package manager to handle projects dependencies
- Use `uv add <library>` when installing a library
- Use Python 3.12 as default Python version

## Library choices

- Use matplotlib for data visualization
- Use pandas (or Polars)
- Use numpy for numbers manipulation
- Use scikit-learn for simple and efficient data analysis

## PR instructions

- Title format: [<project_name>] <Title>
- Always check for errors before committing
